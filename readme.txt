Voici les étapes pour exécuter le projet de Gestion de Système Garage V.Parrot

Téléchargez le fichier zip.
Décompressez le fichier et copiez le dossier "garagev".
Collez-le dans le répertoire racine (pour xampp : xampp/htdocs, pour wamp : wamp/www, pour lamp : var/www/html).
Ouvrez PHPMyAdmin (http://localhost/phpmyadmin).
Créez une base de données avec le nom "v.parrotgarage".
Importez le fichier "v.parrotgarage" (fourni dans le dossier SQL du package zip).
Lancez le script http://localhost/garagev (frontend).
Pour le panneau d'administration, accédez à http://localhost/garagev/admin.php.
